Shopify Liquid to HTML Cheat Sheet

A comprehensive guide for both beginners and senior developers

Basic Liquid Syntax

  • {{ "{{ product.title }}" }} - Objects output content
  • {% "{% if %}" %}, {% "{% for %}" %} - Tags control logic
  • {{ "{{ name | upcase }}" }} - Filters modify output
// Basic Object Example
{{ "{{ product.title }}" }}
{{ "{{ product.price | money_with_currency }}" }}
{{ "{{ product.description | truncate: 150 }}" }}

// Basic Tag Example
{% "{% if product.available %}" %}
  In stock!
{% "{% else %}" %}
  Sold out!
{% "{% endif %}" %}

Control Flow

Conditional Statements

// If Statement
{% "{% if condition %}" %}
  // code
{% "{% elsif other_condition %}" %}
  // code
{% "{% else %}" %}
  // code
{% "{% endif %}" %}

// Unless Statement
{% "{% unless condition %}" %}
  // code
{% "{% endunless %}" %}

// Case Statement
{% "{% case handle %}" %}
  {% "{% when 'value1' %}" %}
    // code
  {% "{% when 'value2' %}" %}
    // code
  {% "{% else %}" %}
    // default code
{% "{% endcase %}" %}

Loops and Iterations

// Basic For Loop
{% "{% for product in collection.products %}" %}
  {{ "{{ product.title }}" }}
{% "{% endfor %}" %}

// For Loop with Limit
{% "{% for product in collection.products limit:5 %}" %}
  {{ "{{ product.title }}" }}
{% "{% endfor %}" %}

// For Loop with Offset
{% "{% for product in collection.products offset:5 limit:5 %}" %}
  {{ "{{ product.title }}" }}
{% "{% endfor %}" %}

// Reversed Loop
{% "{% for item in collection.products reversed %}" %}
  {{ "{{ item.title }}" }}
{% "{% endfor %}" %}

Advanced Filters

String Filters

{{ "{{ product.title | upcase }}" }}  // Converts to uppercase
{{ "{{ product.title | downcase }}" }}  // Converts to lowercase
{{ "{{ product.title | capitalize }}" }}  // Capitalizes first letter
{{ "{{ product.title | escape }}" }}  // Escapes HTML
{{ "{{ product.title | strip_html }}" }}  // Removes HTML tags
{{ "{{ product.title | truncate: 20, '...' }}" }}  // Truncates text

Number Filters

{{ "{{ product.price | money }}" }}  // Formats as money
{{ "{{ product.price | money_without_currency }}" }}
{{ "{{ product.price | money_with_currency }}" }}
{{ "{{ 3 | times: 2 }}" }}  // Mathematical operations
{{ "{{ 3 | divided_by: 2 }}" }}
{{ "{{ 3 | plus: 2 }}" }}
{{ "{{ 3 | minus: 2 }}" }}

Working with Arrays

// Array Filters
{{ "{{ product.tags | join: ', ' }}" }}  // Joins array elements
{{ "{{ product.tags | first }}" }}  // Gets first element
{{ "{{ product.tags | last }}" }}  // Gets last element
{{ "{{ product.tags | size }}" }}  // Gets array size
{{ "{{ product.tags | sort }}" }}  // Sorts array
{{ "{{ product.tags | uniq }}" }}  // Removes duplicates
